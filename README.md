# Poradnik i40 D4FD manual

Poradnik właściciela Hyundai i40 1.7 CRDi (silnik D4FD) — zimny rozruch, zasady eksploatacji, serwis, typowe usterki, części i dane techniczne.

## Jak wyświetlić graficznie (w przeglądarce)

Kliknij poniższy link — plik `index.html` zostanie wyrenderowany bezpośrednio w przeglądarce:

**[Otwórz Poradnik](https://htmlpreview.github.io/?https://raw.githubusercontent.com/igor1516/Poradnik-i40-D4FD/main/index.html)**

## Zawartość poradnika

- **Zimny rozruch** — krok po kroku, tabela temperatur, mity vs rzeczywistość
- **Zasady eksploatacji** — sygnały alarmowe, 10 złotych zasad dieslisty
- **Przed zimą** — lista kontrolna, harmonogram serwisowy
- **Serwis i40** — typowe usterki z opisem i kosztami (klapy wirowe, łańcuch rozrządu, turbo VGT, wtryskiwacze, EGR, rdza, woda w bagażniku...)
- **Części i koszty** — numery katalogowe OEM, co kupować w oryginale
- **Dane techniczne** — oleje, płyny, momenty dokręcania
